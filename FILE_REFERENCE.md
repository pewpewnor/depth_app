## 🎯 Model & Calibration

**Model**: Depth-Anything-V2-Small  
**Source**: huggingface.co/LiheYoung/depth-anything-small-hf  
**Export Format**: ONNX (Open Neural Network Exchange)  
**Location**: `assets/models/depth_model.onnx`  

**Calibration Formula**:
```
depth_meters = (raw_depth_value / 147) * 6
```

To recalibrate, measure a known distance and update in `lib/depth_estimator.dart`.

## 🔍 Troubleshooting Reference

| Issue | Solution |
|-------|----------|
| Python 3.10 not found | `uv python install 3.10` |
| Model export fails | `python3.10 export_model.py --verbose` |
| Build fails | `./build.sh clean` then rebuild |
| CUDA not detected | Falls back to CPU automatically |
| Camera permission denied | Grant in app settings |
| Docker build issues | `docker system prune && docker build --progress=plain` |
